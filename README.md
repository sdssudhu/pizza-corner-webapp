# pizza-corner-webapp

PIZZA  CORNER WEB APPLICATION

Technologies used : 
	Front-End
	1. HTML
	2. CSS
	3. JavaScript

	Back-End
	1.MySQL
	2.Apache Tomcat
	3.Java

This project is an online portal to order food from pizza corner(Hypothetical). In this project we have created a login 
through which users can access the cart and buy food and stuff. 
The payment invoice is sent to the customer as a mail using SMTP mail server. 


The project1 folder must be placed inside the web-apps folder of Apache Tomcat.

The database is in sql.txt file which must be added to mysql database.
